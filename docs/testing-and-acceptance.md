# 测试与验收文档

## 当前测试环境

手动验证环境：

- 模拟器：`Pixel_9`
- 应用包名：`com.liuguang.media`
- 基线版本：`1.0.0`
- 已验证 Release 版本：`1.0.0`
- 安装方式：`adb install -r`

CI 验证：

- 工作流：`Android APK`
- 标签构建：`v1.0.0`
- 结果：待本次发布验证
- Release 产物：`Liuguang-v1.0.0.apk`

## 本地构建验收

执行：

```bash
cd Liuguang
./gradlew assembleDebug --stacktrace
```

验收标准：

- 命令成功结束。
- APK 生成于 `app/build/outputs/apk/debug/app-debug.apk`。

## 启动冒烟测试

执行：

```bash
adb install -r Liuguang/app/build/outputs/apk/debug/app-debug.apk
adb shell monkey -p com.liuguang.media -c android.intent.category.LAUNCHER 1
```

验收标准：

- 应用启动不崩溃。
- 底部显示 `片库`、`电视`、`音频`、`我的`。

## 功能验收清单

### 片库

- 片库内容能加载。
- 下拉刷新可以触发重新加载。
- 滚动到底部可以继续加载更多。
- 搜索入口可以进入搜索流程。
- 点击影视卡片可以进入详情页。

### 详情页

- 海报位于左侧。
- 基础信息和简介位于右侧。
- 播放源和播放线路控件紧凑。
- 剧集列表直接显示在当前页面。
- 剧集按钮间距、行高、按钮高度符合紧凑要求。
- 不默认选中第 12 集或其他异常集数。
- 点击剧集可以进入播放页。

### 剧集播放页

- 顶部显示返回按钮和剧集标题。
- 标题后显示当前集数。
- 当前播放源地址可见。
- 点击复制图标可以复制源地址。
- 快退、播放/暂停、快进按钮宽度一致。
- 倍速最高支持 4 倍。
- 全屏播放画面不裁切、不显示不全。
- 分辨率、码率、实时网速尽量准确展示。
- 返回后没有后台声音，没有旧画面残留。

### 电视页

- 默认直播源可以解析频道。
- 搜索频道输入框可过滤频道。
- 换源按钮位于分组行。
- 分组筛选可用。
- 点击频道可以进入直播播放页。

### 音频页

- 可以在电台和播客之间切换。
- 电台列表可以搜索、筛选和播放。
- 播客列表可以搜索、刷新和播放。
- 播放时通知栏和锁屏控制可用。

### 直播播放页

- 顶部风格和剧集播放页一致。
- 顶部显示返回按钮和当前频道名。
- 当前直播源地址可见。
- 点击复制图标可以复制源地址。
- 投屏、播放/暂停、全屏同一行且宽度相等。
- 分辨率、码率、实时网速可展示。
- 返回后没有后台声音，没有旧画面残留。

### 在线播放页

- 从 `我的` 页面可以进入在线播放页。
- 页面显示简短说明文案。
- 播放链接输入框可用。
- 解析播放按钮可用。
- 有效 M3U8 链接可以进入播放页。
- 有效 M3U 链接可以解析频道。
- 无效链接显示错误提示。

### 我的页

- 播放历史入口可打开。
- 在线播放入口可打开。
- 视频源管理入口可打开。
- 直播源管理入口可打开。
- 电台源管理入口可打开。
- 播客源管理入口可打开。
- 重置应用有确认提示，并能恢复默认源。
- 免责声明弹窗可打开。
- 检测更新按钮可触发更新检查。

## 发布更新验收

### 系统覆盖安装验证

已验证命令：

```bash
adb install -r /tmp/liuguang-release-v1.0.0/Liuguang-v1.0.0.apk
adb shell dumpsys package com.liuguang.media | rg "versionCode|versionName"
```

结果：

```text
versionCode=1000000
versionName=1.0.0
```

验收标准：

- `adb install -r` 返回 `Success`。
- 安装后版本号与 Release tag 一致。
- APK 签名和已安装应用签名一致。

### 应用内更新验收

当前应用请求公开 Release 接口：

```text
https://api.github.com/repos/lonnnnnng/liuguang-media/releases/latest
```

接口验收：

- HTTP 状态码为 `200`。
- 返回的最新 Release 版本高于当前安装版本时，设置页弹出更新对话框。
- 对话框展示当前版本、新版本、更新说明和 APK 大小。
- 点击下载后显示进度条。
- 下载完成后唤起 Android 系统安装器。
- 如果当前已是最新版本，设置页提示已是最新版本。

## 回归测试清单

- 本地构建通过。
- GitHub Actions tag 构建通过。
- 推送 `v*` 标签后 Release 生成成功。
- Release APK 存在且大小正常。
- APK 签名 SHA 与预期签名一致。
- Release APK 可以覆盖安装旧版本。
- 更新后应用启动正常，历史数据和数据源不异常丢失。
- 至少验证一个剧集播放、一个直播播放、一个电台或播客播放、一个在线 M3U8 播放。

- 第三方 IPTV M3U 源可用性。
- GitHub Release 可访问性。
- 模拟器或真机网络状态。
