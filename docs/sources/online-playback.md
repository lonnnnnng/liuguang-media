# 在线播放源文档

## 适用模块

“我的 -> 在线播放”用于临时输入一个播放链接，不进入源管理列表。它适合临时测试 M3U8 视频链接或 M3U 直播列表。

## 支持格式

在线播放支持两类：

- 单个 M3U8 视频链接。
- M3U 直播列表链接。

## M3U8 视频链接

示例：

```text
https://example.com/video/index.m3u8
```

输入后点击“解析播放”，应用会进入剧集播放式页面，标题显示为“在线播放”，集数标识为 `M3U8`。

适合：

- 临时播放单个 HLS 视频。
- 测试资源站返回的 m3u8 是否可用。
- 测试 CDN、鉴权参数、清晰度链路。

## M3U 直播列表链接

示例：

```text
https://example.com/live/playlist.m3u
```

输入后点击“解析播放”，应用会按电视直播源逻辑解析频道列表。解析成功后可以选择频道进入直播播放页。

支持标准 M3U：

```text
#EXTM3U
#EXTINF:-1 tvg-name="CCTV-1" group-title="央视",CCTV-1
https://example.com/live/cctv1/index.m3u8
```

也支持简单格式：

```text
CCTV-1,https://example.com/live/cctv1/index.m3u8
```

## 链接判断规则

应用会根据 URL 判断处理方式：

| URL 特征 | 处理方式 |
| --- | --- |
| 包含 `.m3u8` | 作为单个视频链接播放 |
| 以 `.m3u` 结尾或包含 `.m3u?` | 作为直播列表解析 |

如果链接既不是 `.m3u8` 也不是 `.m3u`，会提示不支持。

## 不支持的格式

在线播放当前不支持：

- MP4 直链。
- FLV 直链。
- DASH `.mpd`。
- 网页播放地址。
- 需要网页登录态或复杂请求头的地址。

## 定制建议

- 临时视频播放请使用完整 `.m3u8` 链接。
- 临时直播列表请使用 `.m3u` 链接。
- 如果 M3U 列表中频道较多，建议优先把常用频道放前面。
- 链接如带 token，请确认 token 有足够有效期。
- 在线播放不会监听剪切板，必须手动粘贴并点击解析。
